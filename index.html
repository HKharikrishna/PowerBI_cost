📌 Critical Technical Considerations

  • Dataset Memory Rule: Single dataset cannot exceed 50% of capacity memory for optimal performance.
  • Query Timeout: DirectQuery has 30-second timeout; Live connections have 120-second timeout.
  • Embedding Tokens: Expire after 1 hour; implement token refresh logic in application.
  • API Rate Limits: 200 calls per hour per user for Pro; 200,000 per hour for Premium capacity.
  • Dataflow Limitations: Only available in Premium/Fabric capacities, not in Pro licenses.
  • Export Limitations: PDF/PowerPoint export limited to 50 pages; Excel export limited to 150,000 rows.
  • Streaming Dataset: Limited to 1M rows/hour with 200K row retention in Premium.
  • Mobile Offline: Reports limited to 250 MB for offline viewing.
